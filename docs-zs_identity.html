Overview

Features: name, age, nationality, RP-fields.

Installation

1) put in resources/[aurenix]/zs_identity
2) ensure zs_identity
3) set permissions
4) restart

Config

Config.Locale = 'en'
Config.MinAge = 18
Config.MaxAge = 80

Events

RegisterNetEvent('zs_identity:open')
AddEventHandler('zs_identity:save', function(data) ... end)

Exports

exports['zs_identity']:GetPlayerData(src)
exports['zs_identity']:SetPlayerName(src, name)
