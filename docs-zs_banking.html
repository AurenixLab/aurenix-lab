Overview

Feature overview, requirements (ESX/QBCore, MySQL).

Installation

1) put in resources/[aurenix]/zs_banking
2) ensure zs_banking
3) configure config.lua
4) restart

Config

Config.Locale = 'en'
Config.StartingBalance = 2500
Config.TransferFee = 0.02

Events

// Server
AddEventHandler('zs_banking:deposit', function(src, amount) ... end)
AddEventHandler('zs_banking:withdraw', function(src, amount) ... end)
// Client
RegisterNetEvent('zs_banking:updateBalance', function(balance) ... end)

Exports

exports['zs_banking']:GetBalance(src)
exports['zs_banking']:AddMoney(src, amount)
exports['zs_banking']:RemoveMoney(src, amount)
